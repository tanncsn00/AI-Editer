Always follow the full brainstorming → script → approval flow before rendering any video. Do NOT rush ahead and auto-generate content without user sign-off on the script.

**Why:** User expects to review and approve the script/content direction before any assets are generated. The first n8n-claw explainer followed the right process (narrative structure → approve → visual design → approve → render). The TikTok video skipped all of that and the user called it out.

**How to apply:** For every new video, always:
1. Propose topic/angle
2. Write script with scene breakdown
3. Get user approval on script
4. Then and only then generate assets and render
