# The Review — Stick Teacher Universal Format for TikTok/Reels

Skill này là **preset visual + workflow CHUNG** cho format **"The Review — thầy người que paper-cream"**. Không gắn với 1 series cụ thể nào. Dùng được cho mọi video:

- 🛠️ **Tool/App review** — review 1 phần mềm, SaaS, AI agent, app mobile
- 📚 **Knowledge share** — chia sẻ kiến thức (finance, sức khoẻ, productivity, marketing, đầu tư, tâm lý...)
- 🧑‍🎓 **Tutorial/Setup** — hướng dẫn dùng 1 tool / setup 1 quy trình
- 🆚 **Comparison** — so sánh 2-N options (tool A vs tool B, framework, model AI...)

**Vibe Editing series** chỉ là 1 application reference (Ep1 là ví dụ Tutorial/Setup format).

## 4 FORMAT VARIANT — chọn theo loại video

### Format A — REVIEW (tool / app / SaaS / khoá học / sách / sản phẩm)
**Duration**: 60-90s

| # | Beat | Visual |
|---|------|--------|
| 1 | HOOK + VERDICT FLASH | Big claim 1 câu ("Notion là Google Docs có não") + teacher waving |
| 2 | LÀ GÌ + CHO AI | Pipeline 3 ô (input → tool → output) + ai dùng (badge user types) |
| 3 | TÍNH NĂNG NỔI BẬT | Grid 3-4 card icon + 1 dòng mỗi card |
| 4 | DEMO REAL | Real screen recording badge + speed 2-3x |
| 5 | ƯU - NHƯỢC | 2 cột: ✅ ƯU (3 bullet) vs ❌ NHƯỢC (2 bullet) |
| 6 | GIÁ + AI NÊN DÙNG | Card price tier + recommendation cụ thể |
| 7 | VERDICT + SCORE | Big number (vd "8/10") + 1 câu kết + teacher thumbs |
| 8 | OUTRO | CTA follow + link bio + tease video next |

### Format B — TUTORIAL / SETUP (hướng dẫn step-by-step)
**Duration**: 90-180s

| # | Beat | Visual |
|---|------|--------|
| 1 | HOOK | 2 card so sánh CÁCH CŨ (X đỏ) vs CÁCH MỚI (⚡), teacher waving |
| 2 | LÀ GÌ | Pipeline 3 ô (input → AI/tool → output), title "___ LÀ GÌ?" |
| 3 | CHỌN GÌ | Grid 2×3 options, 1-2 ô gold "RECOMMEND" + sao vàng |
| 4 | SETUP / CLONE | Prereq icon row + terminal mock typewriter card |
| 5 | CONFIG / KEYS | Card .env hoặc settings — key đỏ "BẮT BUỘC" + key xanh "TÙY CHỌN" |
| 6 | DEMO REAL | Real screen recording badge + speed 2-3x |
| 7 | KẾT QUẢ REAL | Real screen recording mở output + zoom 2x |
| 8 | OUTRO + EP-NEXT | Grid đồ chơi chưa khoe + tease tập sau + credit + CTA |

> **Reference**: `projects/vibe-editing-ep1/final.mp4` là Format B mẫu.

### Format C — KNOWLEDGE SHARE / EXPLAINER (chia sẻ kiến thức / giải thích concept)
**Duration**: 60-120s

| # | Beat | Visual |
|---|------|--------|
| 1 | HOOK + VẤN ĐỀ | Câu hỏi đắt giá ("Tại sao 90% người sai khi ___?") + teacher pointing |
| 2 | CONTEXT | Card timeline / số liệu / story ngắn |
| 3 | CORE POINT 1 | Big title + card minh hoạ |
| 4 | CORE POINT 2 | Big title + card minh hoạ |
| 5 | CORE POINT 3 | Big title + card minh hoạ (3 là số magic; 2-4 đều ok) |
| 6 | VÍ DỤ THỰC TẾ | Real screen recording / case study / 1 panel cartoon minh hoạ |
| 7 | TAKEAWAY | 1 câu chốt vàng + teacher thumbs + BigWord drop |
| 8 | OUTRO + CTA | Follow + comment bait + link reference (sách/khoá/repo) |

### Format D — COMPARISON (so sánh 2-N options)
**Duration**: 90-150s

| # | Beat | Visual |
|---|------|--------|
| 1 | HOOK + STAKES | "X vs Y — chọn sai mất 6 tháng" + teacher waving |
| 2 | OPTION A INTRO | Card lớn + 3 bullet đặc trưng |
| 3 | OPTION B INTRO | Card lớn + 3 bullet đặc trưng (+ Option C/D nếu có) |
| 4 | DEMO REAL A | Real screen recording option A |
| 5 | DEMO REAL B | Real screen recording option B |
| 6 | BẢNG SO SÁNH | Table 2-N cột × 4-6 hàng, ô gold cho winner mỗi hàng |
| 7 | CHỌN AI? KHI NÀO | Decision tree mini ("nếu X → A, nếu Y → B") + teacher pointing |
| 8 | VERDICT + OUTRO | Final pick + caveat + CTA + tease |

## GATED WORKFLOW — 8 bước. KHÔNG SKIP.

### BƯỚC 1 — CHỌN FORMAT + TOPIC + PERSONA
- Pick 1 trong 4 format (A/B/C/D) theo bản chất video.
- Topic cụ thể (vd: "review Cursor", "chia sẻ 3 cách tiết kiệm tiền", "tutorial Claude Code setup", "Cursor vs Claude Code").
- **Persona** (chọn 1):
  - **"Chồng dạy vợ"** — gần gũi, đời thường, KHÔNG jargon (default cho Vibe Editing series)
  - **"Anh Hai chia sẻ"** — chân thành, khuyến khích (default cho life/finance/health knowledge)
  - **"Thầy giáo gen-Z"** — dí dỏm, slang nhẹ (cho audience trẻ, Gen-Z)
  - **"Senior dev review"** — pro, có opinion, nói thẳng (cho dev/tech audience)
  - **"Tư vấn viên"** — neutral, dữ liệu (cho comparison/decision-making)
- Đặt tên video: `<Topic> — <Hook 1 câu>`. KHÔNG show "Tập X" trong thumbnail (per `feedback_general_no_tap_label`).
- **GATE 1**: Confirm format + topic + persona với user trước khi qua bước 2.

### BƯỚC 2 — BRAINSTORM 8 BEAT
- Fill scaffold của format đã chọn với content cụ thể.
- Mỗi beat 1 dòng outline + visual ý tưởng.
- Xác định beat nào dùng real screen recording / demo / case study.
- **GATE 2**: User duyệt beat sheet trước khi viết script TTS.

### BƯỚC 3 — VIẾT SCRIPT TTS
- Length theo format: REVIEW 250-400 từ, TUTORIAL 450-600 từ, EXPLAINER 200-350 từ, COMPARISON 350-500 từ.
- **Persona LOCK** theo persona đã chọn (consistent từ đầu đến cuối).
- Hook đầu **<5s** — câu đắt giá (claim/question/stat).
- Mỗi beat 1-2 câu, transition mượt.
- Tránh từ tiếng Anh đứng 1 mình — chèn giải thích VN ngay sau ("Notion — phần mềm ghi chú thông minh").
- **TUÂN THỦ `feedback_tts_vn_short_syllables`**: tránh syllable VN cụt như "ý", "ờ", "ạ" đứng 1 mình → ghép cụm.
- Kết bằng CTA cụ thể (follow / comment / link bio / save).
- **GATE 3**: PHẢI gửi user duyệt script TRƯỚC khi chạy TTS + render asset (per `feedback_general_approve_script`).

### BƯỚC 4 — CHUẨN BỊ ASSET (real-screen optional theo format)
- **Format A/B/D có DEMO REAL**: record screen tool/app — speed 2-3x bằng ffmpeg `setpts=PTS/2.5` — letterbox pad 1080×1920 black bar.
- **Format C có thể skip real-screen**: dùng cartoon panel minh hoạ / chart / number animation thay thế.
- Asset image bổ sung: icon set, logo brand (nếu review tool), chart/graph (nếu data-driven).
- Lưu vào `projects/<slug>/clips/` + `projects/<slug>/assets/`.

### BƯỚC 5 — RUN TTS + WHISPER ALIGN
- TTS qua EverAI voice đã chọn (theo persona) — split per beat hoặc full.
- **BẮT BUỘC** `feedback_caption_whisper_verify`: chạy faster-whisper với `initial_prompt=script` rồi DIFF với script. Sửa tay sai sót VN.
- **PHONETIC RULE** `feedback_caption_phonetic_display`: nếu TTS dùng phonetic ("vai bờ ê đít tinh"), caption hiển thị form chuẩn ("vibe editing") — merge sequences trong words.json.
- Output: `tts/voice_full.mp3` + `tts/beat_N.mp3` + `words.json` + `beats.json`.

### BƯỚC 6 — BUILD REMOTION COMPOSITION
- **Template base**: copy `remotion-composer/src/VibeEditingEp1.tsx` → `<NameSlug>.tsx`.
- Reuse component LOCK (KHÔNG viết lại từ đầu):
  - `<Bg/>` paper-cream + noise + vignette
  - `<Teacher/>` (props: `pointing` / `waving` / `thumbs` / `idle` / mouthOpen)
  - `<TitleBig/>` `#E85838` accent uppercase
  - `<Card/>` cream-white border ink
  - `<TerminalMock/>` mac dot + dark bg + typewriter (Format A/B/D)
  - `<RealBadge/>` top-right pill 🔴 REAL · ___
  - `<Letterbox/>` cho real footage
  - `<Caption/>` + `<BigWord/>` block từ `ImLangFull.tsx` verbatim (emphasis `#E5A53B`)
- Sửa SCENES/BEATS/WORDS theo episode + format-specific blocks (table cho D, score badge cho A, takeaway frame cho C).
- Register vào `Root.tsx` với id `<NameSlug>`.
- Render thumbnail composition riêng `<NameSlug>Thumbnail.tsx` (KHÔNG frame extract).

### BƯỚC 7 — RENDER + VERIFY
- `npx remotion render <NameSlug> projects/<slug>/draft.mp4`.
- Preview 5-7 frames đại diện cho user review.
- Iterate theo feedback (pose teacher, timing, color, beat trim).
- Final render khi user duyệt: `final.mp4`.

### BƯỚC 8 — DELIVERABLES (MANDATORY)
**Theo `feedback_general_deliverables`:**
- ✅ `projects/<slug>/final.mp4` — 9:16 1080×1920 @ 30fps
- ✅ `projects/<slug>/thumbnail.png` — designed thumbnail (NOT frame extract): title cam + hook 1 dòng + visual key + CTA
- ✅ `projects/<slug>/caption.md`:
  - Hook ngắn (1 câu < 60 char) + 2-3 dòng body + CTA + 6-8 hashtag VN
  - Pin comment với link/credit (nếu có)
  - 2-3 alt hook A/B test
  - Scene breakdown table (8 beat)
  - Voice + format + persona metadata

## TONE CHEAT-SHEET — Dos / Don'ts

### ✅ DOS (mọi format)
- Hook đắt: claim mạnh / câu hỏi / stat shock ("90% người dùng Notion sai cách")
- Ví dụ đời sống thay jargon ("như mở khoá điện thoại bằng vân tay" thay "auth flow")
- Câu ngắn, chủ-vị-tân rõ
- BigWord drop CHỈ ở HOOK, KẾT, VERDICT (không spam giữa beat)
- Persona consistent từ đầu đến cuối
- CTA cụ thể ("Comment X bạn dùng tool gì" hơn "Theo dõi mình nhé")

### ❌ DON'TS
- ~~"Hãy cùng tìm hiểu về..."~~ (giáo điều)
- ~~"Trong tập này, chúng ta sẽ deep-dive vào..."~~ (sáo)
- ~~"API key, env variable, package.json"~~ (jargon trần trụi không giải thích)
- ~~"Tập 1: Setup môi trường"~~ (show "Tập X" trong thumbnail)
- ~~Big-word ngẫu nhiên giữa beat~~ (gây loãng)
- ~~Persona lẫn lộn~~ (nửa serious dev, nửa chồng-vợ)
